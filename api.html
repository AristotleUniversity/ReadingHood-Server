ReadingHood Server API

The ReadingHood server uses RESTful API. All actions are performed with HTTP requests.
Every request (except register) needs authentication with email and password of the ReadingHood user.
To create a link you should start with: https://localhost:8443/
Then you add the parameters according to the instructions below.

Register

It is used for registering a user to the ReadingHood Server.
Mapping: /register
Parameters:
  • username
  • email
  • password
  • name (optional)
  • surname (optional)
  • department (optional)
Example: https://localhost:8443/register?username=test_user&email=test@mail.gr&password=test_password&name=Test&surname=User&department=AUTH
Result: OK

Profile

Mapping: /profiles

Functions

  • /all
    Returns all the profiles
    Example: https://localhost:8443/profiles/all
    Result: 
    [
  {
	"id": 1,
	"name": "First",
	"surname": "Account",
	"department": "CSD",
  },
  {
	"id": 2,
	"name": "Test",
	"surname": "User",
	"department": "AUTH",
  }
]
  • /searchId
    Searches profile with the given id
    Parameter:
    • id
    Example: https://localhost:8443/profiles/searchId?id=2
    Result: 
    {
	"id": 1,
	"name": "First",
	"surname": "Account",
	"department": "CSD",
}
  • /search
    Searches account according to the given name and surname
    Parameter:
    • name
    • surname
    Example: https://localhost:8443/profiles/search?name=t&surname=us
    Result:

  "name": "Test",
  "surname": "User",
  "department": "AUTH"
}
